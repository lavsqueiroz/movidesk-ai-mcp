# 🤖 Movidesk AI MCP

> **MCP Server** para análise inteligente de tickets do Movidesk integrado com SuperDoc

[![Status](https://img.shields.io/badge/status-planning-blue)](https://github.com/lavsqueiroz/movidesk-ai-mcp)

---

## 📋 Sobre o Projeto

Sistema de análise automática de tickets do Movidesk em 3 níveis:

- **N1 - Validação**: Verifica se ticket tem todas informações necessárias
- **N2 - Classificação**: Classifica como Defeito ou Evolutiva (via SuperDoc)
- **N3 - Sugestão**: Sugere correção técnica (apenas defeitos)

**Resultado**: Nota interna no ticket para analista avaliar
